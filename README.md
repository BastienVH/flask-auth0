Flask-Auth0 is an extension for Flask that allows you to authenticate using the Auth0 service.

Compatibility
=============

Flask-Auth0 requires Python 2.7 or 3.4+.


Installation
============

You can install Flask-Auth0 with pip:

```
$ pip install flask-auth0
```
